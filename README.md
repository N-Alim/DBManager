# BookmarksManager

Alim NOUIRA

## Présentation
Projet de gestionnaire de bases de données

## Installation

1. Installez les dépendances npm
```
npm install
```

## Démarrage

1. Déposez le projet dans le dossier xampp/htdocs

2. Activez Apache et MySQL dans le panneau de contrôle de XAMPP

3. Allez à l'url suivante 
```
http://localhost/DBManager/
```
